# StockSage AI

StockSage is an AI-powered educational platform designed to help users better understand the stock market through structured insights and learning tools.

This project is focused on education only and does not provide financial advice.

## What This Is

- An educational stock market platform powered by AI
- Helps users explore stocks, fundamentals, news, and market concepts
- Designed to improve financial literacy and decision-making skills

The platform emphasizes clarity, structure, and responsible use.

## What This Is Not

- Not a trading platform
- Not financial or investment advice
- Not an open-source product

## Features

- Watchlists and portfolio tracking
- AI-generated educational insights
- Fundamentals and market context
- News and sentiment analysis
- Learning-focused tools and resources
